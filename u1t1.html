Aspectos de un Algoritmo

Un algoritmo es una secuencia de pasos ordenados y finitos diseñados para resolver un problema. Los elementos principales incluyen entrada, proceso y salida. Un buen algoritmo debe ser preciso, eficiente y tener una duración finita.
Existen dos tipos de algoritmos cualitativos y cuantitativos, los cualitativos se desarrollan paso a paso el planteamiento de un problema.
Los cualitativos desarrollan los pasos detalladamente implementando calculos matematicos.
Ejemplo de Algoritmo Simple:
Algoritmo para hacer una taza de té:
1.Calentar el agua.
2.Colocar una bolsita de té en una taza.
3.Verter el agua caliente en la taza.
4.Esperar 3 minutos.
5.Retirar la bolsita y disfrutar.
Si bien este ejemplo puede parecer muy simple nos puede ayudar para comenzar a comprender que es lo que representa un algoritmo
posteriormente requeriras de realizar el desarrollo de algoritmos mucho mas detallados de cada paso a seguir para lograr un objetivo.
